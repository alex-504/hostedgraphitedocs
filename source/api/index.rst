API Guides
==========

We have developed a variety of APIs for you to easily build your own integrations to Hosted Graphite tools:

- | :doc:`Metrics API </api/metric_api>`
  | Search account metrics

- | :doc:`Tag API </api/tag_api>`
  | Search tagged metrics

- | :doc:`Render API </api/render_api>`
  | Get more out of your metric data

- | :doc:`Render Variables API </api/variables_api>`
  | Build dynamic metric queries

- | :doc:`Dashboard API </api/dashboard-api>`
  | Fetch, create and update your dashboards

- | :doc:`Annotations API </api/annotations-and-events>`
  | Create events with tags

- | :doc:`Aggregates API </api/aggregates_api>`
  | Consolidate large sets of metrics to improve query retreival

- | :doc:`Alert APIs </alerting/index>`
  | Manage alerts, notification channels, and define scheduled mute periods
